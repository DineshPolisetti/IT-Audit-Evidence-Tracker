# IT Audit Evidence Tracker & Dashboard (Excel)

## Overview
This project is an Excel-based IT Audit Evidence Tracker designed to track audit controls, evidence collection status, findings, severity, and closure timelines.

## Features
- Dropdown validation (Status, Evidence Received, Severity)
- Conditional formatting for pending evidence and high severity
- KPI summary dashboard
- Pie chart for audit status distribution
- Column chart for severity distribution

## Tools Used
- Microsoft Excel
- Data Validation
- Conditional Formatting
- COUNTIF / COUNTA formulas
- Pivot-style reporting and charts

## Output
The dashboard helps audit teams track progress and identify high-risk issues efficiently.
